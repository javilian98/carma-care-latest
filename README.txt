Step 1: Install dependencies
Installation required for this project:
- Node.JS ()

------------------------------------------------------------------

Step 2: Project Setup
Project setup to run the application after installing Node.JS:

Check if there is node_modules folder:
- If have, move to Step 3.

- Else:
  In the carma-care-latest folder, open a terminal and run the command 'npm install' to install the dependencies

-------------------------------------------------------------------

Step 3: Running the app
Once all dependencies are installed:
  a) run the command 'npm run dev', and localhost:5173 will be given for you to access in your browser

  b) Alternatively if you want to run on a built version, 'npm run preview', and localhost:4173 will be given instead
